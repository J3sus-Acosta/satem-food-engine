# Prompts de Generación de Imágenes — SATEM Food Engine

Este documento contiene la guía de generación de prompts para recrear o extender el **Catálogo Oficial de Imágenes de Productos** mediante IA (Midjourney, DALL-E 3, Stable Diffusion, Imagen 3).

---

## Estilo Visual Maestro (Master Style Prompt)

Para asegurar coherencia fotográfica absoluta en todo el menú, todos los prompts extienden el siguiente modificador de estilo:

> **Master Style Prefix:**  
> `Hyperrealistic professional food photography, 45-degree angle shot, centered single item, soft natural studio morning light, light natural oak wooden table background, neutral tones, 8k resolution, crisp detail, cinematic depth of field, no logos, no text, no hands, no people, photorealistic.`
